아래 내용을 바탕으로 **의사 대상 서비스 소개서**를 만들어줘.

## 요구사항
- 대상: 정형외과 개원의 / 봉직의
- 톤: 전문적이면서 읽기 쉽게. 의사가 30초 안에 핵심을 파악할 수 있도록.
- 분량: A4 기준 4~5페이지
- 형식: 깔끔한 문서 형태. 표, 아이콘(이모지), 시각적 구분 적극 활용.
- 언어: 한국어 (의학 용어는 영문 병기)

## 서비스 개요

**서비스명:** Dr.Note  
**한줄 소개:** 진료 대화를 녹음하면, AI가 차트 + 보험코드 + 처방전까지 한번에 완성합니다.  
**타겟:** 정형외과 (향후 다과 확장 예정)  
**핵심 가치:** 의사의 차트 작성 시간을 71% 절감

## 포함할 내용 (이 순서대로)

### 페이지 1: 표지 + 핵심 메시지

- 서비스 로고/이름: Dr.Note
- 태그라인: "의사의 시간을 환자에게 돌려드립니다"
- 한줄 소개
- Before/After 핵심 수치:
  - Before: 환자 1명당 차트 업무 13분 (차트 5분 + 코드 검색 4분 + 처방 3분 + 상담사 전달 1분)
  - After: 환자 1명당 14초 (녹음 종료 → 자동 생성 10초 → EMR 복사 2초 → 다음 환자 2초)
  - 하루 30명 기준: 9시간 → 2시간 37분 (71% 절감)

### 페이지 2: 핵심 기능 (상단 3개)

**1. AI 차트 자동 생성**
- 환자와 대화하면서 녹음 → 종료 즉시 차트 완성
- OpenAI Whisper(음성인식) + GPT-4o(차트 생성)
- 정형외과 전문 약어 자동 적용: HNP, ORIF, ROM, LE, RCS, ACL, MCL 등
- 출력: CC(주호소) / PI(현병력) / Dx(진단) / Plan(치료계획) / Note
- 초진/재진 자동 구분
- 포맷 자유 커스터마이징 (필드 on/off, 순서, 라벨, AI 지시사항)

**2. 보험 청구 코드 자동 추천**
- 진단(Dx)에서 KCD 상병코드 자동 추천 (45개 정형외과 코드)
- 치료계획(Plan)에서 EDI 수가코드 자동 추천 (35개 코드)
- 예시: r/o LE → M77.1 외측상과염 + MX032 체외충격파 + LA313 팔꿈치주사
- 원클릭 복사 → EMR 붙여넣기

**3. 스마트 처방 추천**
- 진단 기반 처방 세트 자동 추천 (13개 질환)
- 약명(한/영), 용량, 용법, 기간, 카테고리 표시
- 나만의 처방 세트 즐겨찾기 저장
- 추가 치료 Tip 포함 (예: "ESWT 2~3주 병행 시 효과적")

### 페이지 3: 편의 기능 (나머지 7개)

**4. EMR 통합 복사**
- 차트 + KCD + EDI + 처방을 한번에 복사하는 "EMR 복사" 버튼
- 하단 고정 진료 마무리 바: [EMR 복사] [차트 복사] [환자 안내] [다음 환자]

**5. 환자 타임라인 & AI 경과 요약**
- 환자별 방문 기록 자동 그룹핑 (이름 or 부위 기반)
- 방문 간 호전/악화 자동 감지
- "경과 요약" 버튼: GPT가 다회차 방문 경과를 자동 요약 (호전도, 타임라인, 권고)
- 소견서/보험심사 대응에 활용

**6. 환자명 자동완성 & 재진 자동 감지**
- 2글자 입력 → 기존 환자 자동완성
- 선택 시: 재진 전환 + 이전 PMH/약물/알러지 자동 로드 + 이전 기록 GPT 주입

**7. 다국어 번역 (영/중/베/일)**
- 외국인 환자 응대용 차트 자동 번역

**8. 환자 교육 자료 자동 생성**
- 진단명에서 환자가 이해할 수 있는 쉬운 설명 생성

**9. 상담사 실시간 연동**
- 의사가 차트 생성 → 상담사 화면에 실시간 표시
- 상담 안내 사항, 보험코드, 처방 모두 자동 전달

**10. AI 영상 분석 & DICOM 뷰어**
- X-ray 영상 AI 분석 (골연령, 척추정렬, 무릎정렬, 일반)
- PACS 서버 연동 DICOM 뷰어

### 페이지 4: 워크플로우 다이어그램 + 보안

**의사의 진료 흐름:**
```
환자 선택 (자동완성) → 녹음 → AI 차트+코드+처방 → EMR 복사 → 다음 환자
          ↓                              ↓
    이전 기록 자동 참조           상담사 실시간 수신
    재진 자동 감지               환자 교육자료 생성
```

**기술 스택:**
| 항목 | 기술 |
|------|------|
| 음성인식 | OpenAI Whisper |
| 차트/번역/교육 | GPT-4o |
| 영상 분석 | GPT-4o Vision |
| 프론트엔드 | Next.js / React / TypeScript |
| DB | Supabase (PostgreSQL) |
| 실시간 | Supabase Realtime |

**보안:**
- TLS 암호화
- Row Level Security (RLS)
- API 키 서버 사이드 관리
- 데이터 소유권: 병원

### 페이지 5: 도입 안내 + CTA

- 현재 정형외과 전문 최적화
- 향후 내과, 재활의학과, 신경외과 등 확장 예정
- 도입 문의 연락처: [여기에 연락처 입력]
- 클로징 문구: "Dr.Note — 의사의 시간을 환자에게 돌려드립니다."

## 디자인 지시

- 색상: 메인 틸(#0D9488), 보조 퍼플(#7C3AED), 배경 화이트
- 각 기능마다 관련 이모지 아이콘 사용
- 표는 깔끔하게, 줄무늬 배경
- Before/After는 빨강(Before) vs 초록(After)으로 대비
- 핵심 수치(71% 절감, 10초 등)는 크고 굵게 강조
- 페이지 하단마다 서비스명 + 태그라인 반복

## 추가 지시

- 의사가 가장 관심 가질 포인트: "시간 절약", "보험 코드 자동", "내 스타일 처방 저장"
- 기술 용어보다는 실제 사용 시나리오 중심으로 설명
- "AI가 대신 해준다"보다 "의사의 판단을 보조한다"는 톤으로
- 차트 예시를 하나 넣어줘:
  - 입력: "팔꿈치가 2주 전부터 아파요. 테니스 치다가 그런 것 같아요."
  - 출력: CC: Rt elbow pain / PI: 2주 전 테니스 후 발생 / Dx: r/o LE / Plan: ESWT, inj, PT
  - 자동 추천: KCD M77.1 / EDI MX032, LA313 / 처방: 셀레콕시브 200mg
