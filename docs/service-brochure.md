# Dr.Note 서비스 소개서

## 의사를 위한 AI 진료 보조 시스템

---

## 한줄 소개

> **진료 대화를 녹음하면, AI가 차트 + 보험코드 + 처방전까지 한번에 완성합니다.**

---

## Dr.Note가 해결하는 문제

| 기존 방식 | Dr.Note |
|-----------|---------|
| 진료 후 차트를 수기로 작성 (5~10분) | 녹음 종료 즉시 차트 자동 생성 (10초) |
| KCD/EDI 코드를 직접 검색 | 진단에서 보험코드 자동 추천 |
| 처방을 매번 수동 입력 | 진단 기반 처방 자동 추천 + 나만의 처방 세트 |
| 재진 환자의 이전 기록을 직접 확인 | 이전 기록 자동 참조, 경과 자동 비교 |
| 외국인 환자에게 설명하기 어려움 | 4개국어 자동 번역 (영/중/베/일) |
| 상담사에게 구두로 전달 | 상담사 화면에 실시간 자동 연동 |

---

## 핵심 기능

### 1. AI 차트 자동 생성

환자와의 대화를 녹음하면, OpenAI Whisper가 음성을 텍스트로 변환하고 GPT-4o가 정형외과 전문 차트를 자동 생성합니다.

**출력 형식:**
- **CC** (Chief Complaint) — 주호소
- **PI** (Present Illness) — 현병력
- **Dx** (Diagnosis) — 진단
- **Plan** — 치료계획
- **Note** — 추가 메모

**특징:**
- 정형외과 전문 약어 자동 적용 (HNP, ORIF, ROM, LE, RCS 등)
- 초진/재진 자동 구분
- CC/PI/Dx/Plan 포맷 자유롭게 커스터마이징 가능
- 필드별 AI 지시사항 설정 (예: "영문 약어 위주로 작성")

---

### 2. 보험 청구 코드 자동 추천 (KCD / EDI)

차트가 생성되면, 진단(Dx)에서 **KCD 상병코드**, 치료계획(Plan)에서 **EDI 수가코드**를 자동으로 추천합니다.

| 구분 | 수록 코드 | 예시 |
|------|-----------|------|
| **KCD 상병코드** | 45개 (정형외과 주요) | M77.1 외측상과염, M51.1 요추간판탈출증 |
| **EDI 수가코드** | 35개 (처치/검사/물리치료/주사/영상) | MX032 체외충격파, LA312 어깨관절주사 |

- 코드 클릭 한 번으로 복사 → EMR에 바로 붙여넣기
- 전체 코드 일괄 복사 기능

---

### 3. 스마트 처방 추천

진단에 맞는 처방 세트를 자동으로 추천합니다.

**수록 처방 세트: 13개 질환**

| 질환 | 약물 예시 |
|------|-----------|
| 외측상과염 (테니스엘보) | 셀레콕시브 200mg, 케토프로펜 패치 |
| 요추간판탈출증 (허리디스크) | 셀레콕시브, 프레가발린, 에페리손, 레바미피드 |
| 무릎 퇴행성관절염 | 아세클로페낙, 글루코사민, 케토프로펜 패치 |
| 회전근개 질환 | 아세클로페낙, 에페리손, 디클로페낙 겔 |

**포함 정보:** 약명(한/영), 용량, 용법, 기간, 카테고리, 추가 치료 Tip

**나만의 처방 세트(즐겨찾기):**
- 자주 쓰는 처방 조합을 ★ 즐겨찾기로 저장
- 다음 진료 시 같은 진단이면 자동으로 상단에 표시
- 병원마다, 의사마다 다른 처방 패턴 반영

---

### 4. EMR 통합 복사

차트 작성이 끝나면, 하단 **진료 마무리 바**에서 원클릭으로:

| 버튼 | 복사 내용 |
|------|-----------|
| **EMR 복사** | 차트 + KCD코드 + EDI코드 + 처방 전체 |
| **차트 복사** | CC/PI/Dx/Plan만 |
| **환자 안내** | 환자 교육 자료 |
| **다음 환자** | 초기화 + 새 녹음 준비 |

EMR 시스템에 **붙여넣기 한 번**으로 입력 완료.

---

### 5. 환자 타임라인 & 경과 자동 요약

**환자별 방문 기록 자동 그룹핑:**
- 환자명 또는 진료 부위 기반으로 자동 분류
- 방문 횟차별 타임라인 시각화
- 이전 방문 대비 호전/악화 자동 감지

**AI 경과 요약:**
- 2회 이상 방문 시 "경과 요약" 버튼 클릭
- GPT가 전체 방문 경과를 자동 요약:
  - 치료 타임라인
  - 호전도 평가
  - 향후 치료 권고
- 소견서, 보험 심사 대응에 즉시 활용

---

### 6. 환자명 자동완성 & 재진 자동 감지

- 환자명 **2글자 입력** → 기존 환자 목록에서 자동완성
- 기존 환자 선택 시:
  - ✅ 자동으로 **재진 모드** 전환
  - ✅ 이전 **PMH / 수술력 / 약물 / 알러지** 자동 로드
  - ✅ 이전 **진단/치료 기록**을 GPT에 자동 전달 → 맥락 있는 차트 생성

---

### 7. 다국어 번역 & 환자 교육

**4개국어 자동 번역:**
- 영어 (English)
- 중국어 (中文)
- 베트남어 (Tiếng Việt)
- 일본어 (日本語)

**환자 교육 자료 자동 생성:**
- 진단명 클릭 → 환자가 이해할 수 있는 쉬운 설명 자동 생성
- 포함: 질환 설명, 원인, 증상, 치료법, 주의사항, 회복기간

---

### 8. 상담사 실시간 연동

의사가 차트를 생성하면, 접수/수납 데스크의 **상담사 화면**에 실시간으로 전달됩니다.

**상담사가 보는 정보:**
- 진료 차트 (CC/PI/Dx/Plan)
- 상담사 안내 사항 (의사 설명, 치료 이유, 치료 항목)
- 보험 청구 코드 (KCD/EDI)
- 추천 처방 세트
- 새 기록 도착 시 실시간 알림

→ 의사가 별도로 설명할 필요 없이, 상담사가 자체적으로 환자 안내 가능.

---

### 9. AI 영상 분석 (GPT-4o Vision)

정형외과 X-ray 영상을 AI가 분석합니다.

| 분석 유형 | 설명 |
|-----------|------|
| **골연령 분석** | 소아 성장판 평가 |
| **척추 정렬** | 척추 만곡, 디스크 간격 분석 |
| **무릎 정렬** | 내반/외반 각도, 관절 간격 |
| **일반 분석** | 골절, 탈구, 기타 소견 |

---

### 10. DICOM 뷰어 (PACS 연동)

- 병원 PACS 서버와 직접 연동
- DICOM 파일 직접 열기
- 밝기/대비(Window/Level) 조절
- 확대/축소/회전
- 영상 캡처 → AI 분석 연동

---

## 차트 포맷 커스터마이징

각 의사의 스타일에 맞게 차트 포맷을 자유롭게 설정할 수 있습니다.

**설정 가능 항목:**
- 필드 표시/숨기기 (CC, PI, Dx, Plan, Note)
- 필드 순서 드래그 앤 드롭으로 변경
- 뱃지, 라벨, 색상 커스터마이징
- 커스텀 필드 추가 (원하는 항목 자유 추가)
- 필드별 AI 지시사항 (예: "한 줄로 요약", "영문 약어만")
- 전체 추가 지시사항 (예: "모든 진단에 r/o 붙이기")
- 설정 자동 저장 → 다음 접속 시 복원

## 기술 스택

| 항목 | 기술 |
|------|------|
| 음성인식 (STT) | OpenAI Whisper |
| 차트 생성 | GPT-4o |
| 영상 분석 | GPT-4o Vision |
| 프론트엔드 | Next.js 16 / React 19 / TypeScript |
| 데이터베이스 | Supabase (PostgreSQL) |
| 실시간 연동 | Supabase Realtime |
| 보안 | RLS (Row Level Security), HTTPS |

---

## 보안 & 개인정보

- **데이터 암호화**: 전송 중 TLS 암호화
- **접근 제어**: Row Level Security (RLS) 적용
- **API 키 관리**: 서버 사이드에서만 처리, 클라이언트 노출 없음
- **데이터 소유권**: 모든 데이터는 병원 소유

## 의사의 하루 (Before vs After)

### Before Dr.Note
```
환자 진료 (5분)
→ 차트 수기 작성 (5분)
→ KCD 코드 검색 (2분)
→ EDI 코드 검색 (2분)
→ 처방 입력 (3분)
→ 상담사에게 구두 전달 (1분)
─────────────────────
1명당 약 18분
하루 30명 = 540분 (9시간)
```

### After Dr.Note
```
환자명 선택 (2초)
→ 녹음 시작/종료 (진료 중 자동)
→ AI 차트 + 코드 + 처방 자동 생성 (10초)
→ EMR 복사 버튼 클릭 (1초)
→ 다음 환자 (1초)
─────────────────────
1명당 약 5분 14초
하루 30명 = 157분 (2시간 37분)
차트 업무 시간 71% 절감
```
