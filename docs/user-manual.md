## 시작하기

### 접속 방법
- PC: 브라우저에서 **사이트 주소** 접속
- 모바일/태블릿: 브라우저 접속 후 **"홈 화면에 추가"** (앱처럼 사용 가능)

### 역할별 페이지

| 역할 | 접속 주소 | 용도 |
|------|-----------|------|
| **의사** | `/demo` | 진료 녹음 → 차트 생성 → EMR 복사 |
| **상담사/간호사** | `/counselor` | 실시간 차트 수신 → 환자 안내 |
| **영상 분석** | `/imaging` | X-ray AI 분석 |

## 의사용 - 진료 기록

### 기본 흐름

```
1. 환자 이름 입력 (2글자 → 자동완성)
2. 초진/재진 선택
3. [녹음 시작] 클릭
4. 환자와 대화 (실시간 자막 표시)
5. [녹음 완료] 클릭
6. AI가 차트 자동 생성 (약 10초)
7. 하단 [EMR 복사] 클릭
8. EMR에 붙여넣기
9. [다음 환자] 클릭
```

### 단계별 상세

#### 1단계: 환자 선택
- **환자 이름칸**에 이름을 입력하세요
- 기존 환자면 **드롭다운에서 선택** → 자동으로 재진 전환 + 이전 정보 로드
- 신규 환자면 이름만 입력하고 **초진** 그대로 진행

#### 2단계: 녹음
- **[녹음 시작]** 버튼을 누르면 마이크가 활성화됩니다
- 환자와 대화하세요 — **실시간으로 자막이 표시**됩니다
- 일시정지가 필요하면 **[⏸]** 버튼
- 대화가 끝나면 **[녹음 완료]** 버튼

#### 3단계: 차트 확인
AI가 자동으로 생성한 차트를 확인합니다:

| 항목 | 설명 |
|------|------|
| **CC** | 주호소 (Chief Complaint) |
| **PI** | 현병력 (Present Illness) |
| **Dx** | 진단 (Diagnosis) |
| **Plan** | 치료 계획 |
| **Note** | 추가 메모 |

- 각 항목 옆 **[복사]** 버튼으로 개별 복사 가능
- 내용 수정이 필요하면 EMR에서 직접 수정

#### 4단계: 보험코드 & 처방 확인
차트 아래에 자동으로 표시됩니다:

- **KCD 상병코드** — 코드 클릭 → 복사
- **EDI 수가코드** — 코드 클릭 → 복사
- **처방 추천** — 약물명, 용량, 용법 확인
- 처방 옆 **[★]** 클릭 → 나만의 처방 세트로 저장

#### 5단계: EMR 복사 & 마무리
화면 하단 고정 바에서:

| 버튼 | 동작 |
|------|------|
| **📋 EMR 복사** | 차트 + 보험코드 + 처방 한번에 복사 |
| **📄 차트 복사** | 차트 내용만 복사 |
| **📱 환자 안내** | 환자 교육 자료 복사 |
| **🎙️ 다음 환자** | 초기화 → 새 녹음 준비 |

### 차트 포맷 커스터마이징

AI Chart 헤더의 **[포맷 설정]** 버튼을 클릭하면:

- **필드 켜기/끄기** — 필요 없는 항목 숨기기
- **순서 변경** — 드래그 앤 드롭으로 순서 조정
- **라벨 수정** — "Chief Complaint" → "주호소" 등
- **AI 지시** — "영문 약어 위주로 작성" 등 필드별 지시
- **커스텀 필드** — [+ 필드 추가]로 원하는 항목 추가

설정은 자동 저장되며 다음 접속 시 유지됩니다.

---

### 다국어 번역

차트 하단 번역 버튼:
- **English** / **中文** / **Tiếng Việt** / **日本語**
- 클릭하면 차트 전체가 해당 언어로 번역됩니다
- 외국인 환자에게 보여주기 용도

---

### 환자 교육 자료

Dx (진단) 영역에서:
- **[환자 설명]** — 해당 진단의 쉬운 설명 생성
- **[전체 분석]** — 교육자료 + 보험코드 + 처방 한번에

---

### 환자 타임라인

상단 메뉴 **[환자 타임라인]** 버튼:
- 환자별 방문 기록 자동 그룹핑
- 방문 간 호전/악화 표시
- **[경과 요약]** 버튼 → AI가 전체 방문 경과 자동 요약
- 소견서 작성 시 활용 가능

## 상담사/간호사용 - 실시간 뷰

### 접속
브라우저에서 `/counselor` 접속 (또는 메인에서 **[상담사]** 클릭)

### 사용법

```
1. 페이지를 열어둡니다
2. 의사가 차트를 생성하면 자동으로 알림이 옵니다
3. 왼쪽 목록에서 기록을 클릭합니다
4. 오른쪽에 상세 내용이 표시됩니다
```

### 상담사 화면에서 볼 수 있는 정보

| 항목 | 내용 |
|------|------|
| **진료 차트** | CC/PI/Dx/Plan |
| **상담 안내** | 의사 설명 요약, 치료 이유, 치료 항목 |
| **보험 코드** | KCD/EDI 코드 (복사 가능) |
| **처방 정보** | 추천 약물, 용량, 기간 |

- 새 기록이 도착하면 **빨간 알림 뱃지**가 표시됩니다
- **[새로고침]** 버튼으로 수동 갱신도 가능

## 앱 설치 방법

Dr.Note는 별도 앱 스토어 다운로드 없이 **홈 화면에 추가**하면 앱처럼 사용할 수 있습니다.

### iPhone / iPad (Safari)

```
1. Safari에서 Dr.Note 접속
2. 하단 공유 버튼 (□↑) 탭
3. "홈 화면에 추가" 선택
4. "추가" 탭
→ 홈 화면에 Dr.Note 아이콘 생성
```

### Android (Chrome)

```
1. Chrome에서 Dr.Note 접속
2. 주소창 오른쪽 ⋮ 메뉴 탭
3. "홈 화면에 추가" 또는 "앱 설치" 선택
4. "설치" 탭
→ 홈 화면에 Dr.Note 아이콘 생성
```

### PC (Chrome / Edge)

```
1. 브라우저에서 Dr.Note 접속
2. 주소창 오른쪽 설치 아이콘(⊕) 클릭
3. "설치" 클릭
→ 바탕화면에 Dr.Note 앱 생성
```

## 기능별 상세 안내

### AI 영상 분석 (`/imaging`)

1. 분석 유형 선택 (골연령 / 척추 / 무릎 / 일반)
2. X-ray 이미지 업로드
3. (선택) 환자 나이 입력
4. **[분석 시작]** 클릭
5. AI가 소견, 측정값, 환자 설명을 생성

### DICOM 뷰어 (`/pacs`)

1. PACS 서버 주소 입력 또는 DICOM 파일 직접 열기
2. 밝기/대비 조절: 마우스 드래그
3. 확대/축소: 마우스 휠
4. 회전: 도구 버튼
5. 영상 캡처 → AI 분석 연동 가능

### 의학 용어 사전 (`/dictionary`)

- 52개 정형외과 주요 용어 수록
- 카테고리별 필터 (어깨, 척추, 무릎 등)
- 검색 기능
- 새 용어 추가/수정 가능

## 자주 묻는 질문

**Q: 녹음 중 자막이 안 나와요**
> Chrome 브라우저를 사용해주세요. Safari는 일부 기능이 제한될 수 있습니다. 마이크 권한을 허용했는지 확인하세요.

**Q: 차트 생성이 안 돼요**
> 인터넷 연결을 확인하세요. 녹음이 너무 짧으면(5초 미만) 생성되지 않습니다.

**Q: 이전 환자 기록은 어디서 보나요?**
> 상단 [기록] 버튼 또는 [환자 타임라인] 버튼을 클릭하세요.

**Q: 차트 포맷을 바꾸고 싶어요**
> AI Chart 옆 [포맷 설정] 버튼에서 자유롭게 커스터마이징 가능합니다.

**Q: 상담사 화면에 실시간으로 안 뜨네요**
> Supabase Realtime 연결이 필요합니다. 관리자에게 문의하세요. 연결 전에도 [새로고침]으로 수동 갱신 가능합니다.

**Q: 모바일에서도 되나요?**
> 네. 브라우저에서 접속하면 모바일에서도 모든 기능을 사용할 수 있습니다. 홈 화면에 추가하면 앱처럼 사용 가능합니다.

**Q: 내 데이터는 안전한가요?**
> 모든 데이터는 암호화되어 전송되며, 병원 소유의 데이터베이스에 저장됩니다.
